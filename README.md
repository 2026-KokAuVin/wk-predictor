# ⚽ WK 2026 AI Voorspeller

Een interactieve webapplicatie die Claude AI gebruikt om WK 2026 wedstrijduitslagen te voorspellen op basis van uitgebreide statistieken.

## 🚀 Features

- **5 wedstrijden selecteren** uit de komende WK 2026 groepsfasewedstrijden
- **Claude AI analyse** van elke wedstrijd met meerdere statistieken:
  - FIFA Ranking vergelijking
  - Recente vorm (laatste 5 wedstrijden)
  - Doelsaldo & offensieve kracht
  - Verwachte Goals (xG)
  - Balbezit voorspelling
  - Winstkansen (thuis/gelijk/uit)
- **Tactische analyse** per wedstrijd in het Nederlands
- **Sleutelfactoren** en uitblinkers per wedstrijd
- **Betrouwbaarheidsscore** per voorspelling

## 🛠️ Tech Stack

- Vanilla HTML/CSS/JavaScript (geen dependencies)
- Anthropic Claude API (`claude-sonnet-4-6`)
- Google Fonts (Bebas Neue, Inter, JetBrains Mono)

## 📋 Gebruik

1. Open `index.html` in je browser
2. Selecteer precies **5 wedstrijden**
3. Klik op **AI Voorspelling Starten**
4. Bekijk de gedetailleerde analyse per wedstrijd

> **Note:** De app maakt gebruik van de Anthropic API. Bij lokaal gebruik is een API-sleutel nodig die via de claude.ai interface wordt verstrekt.

## 📊 Statistieken per wedstrijd

| Statistiek | Beschrijving |
|---|---|
| xG (Expected Goals) | Verwachte doelpunten op basis van kansrijkheid |
| Winstkansen | % kans op thuis/gelijk/uitzege |
| Balbezit | Verwacht balbezit percentage |
| Schoten op doel | Verwacht aantal pogingen |
| Recente vorm | Resultaten laatste 5 interlands |
| FIFA Ranking | Actuele wereldranglijst positie |
